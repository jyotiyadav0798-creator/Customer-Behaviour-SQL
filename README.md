# 💻 Customer Behaviour Analysis using SQL

## 📌 Project Overview

This project analyzes customer purchasing behavior using SQL to generate business insights that support data-driven decision-making.

---

## 🎯 Project Objectives

- Analyze customer purchasing patterns.
- Identify top-selling products.
- Evaluate customer spending behaviour.
- Analyze Gold Member contribution.
- Generate business insights using SQL queries.

---

## 🛠 Tools Used

- SQL Server
- SQL Queries
- Relational Database

---

## 📂 Database Tables

The project consists of the following tables:

- Users
- User_Name
- Products
- Sales
- GoldUsers_Signup

---

## 🔍 SQL Concepts Used

- SELECT
- WHERE
- GROUP BY
- HAVING
- ORDER BY
- INNER JOIN
- LEFT JOIN
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions

---

## 📊 Business Questions Solved

- Which products generated the highest revenue?
- Who are the top customers based on spending?
- How much revenue was generated by Gold Members?
- Which customers made the highest number of purchases?
- Which products were purchased most frequently?

---

## 💡 Business Insights

- Premium products generated the highest revenue.
- Gold Members contributed significantly to total sales.
- A small number of customers generated a large share of revenue.
- Customer purchase patterns helped identify loyal customers.

---

## ✅ Business Recommendations

- Focus marketing on high-value customers.
- Strengthen loyalty programs.
- Promote top-performing products.
- Use customer purchase behaviour for targeted campaigns.
